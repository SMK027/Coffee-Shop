Je souhaite créer un site web pour un Coffee Shop.
Il sera séparé en 2 interfaces : une pour les visiteurs et une pour les employés.
L'interface des visiteurs permettra de consulter des pages sans posséder de comptes:
- Page d'accueil avec une présentation du Coffee Shop, des photos et des témoignages de clients.
- Menu avec les différentes boissons proposées, accompagnées de descriptions et de prix
- Formulaire de contact pour des partenariats et diverses questions

L'interface des employés nécessitera une authentification pour accéder aux fonctionnalités suivantes :
- Gestion des commandes : les employés pourront voir les commandes en cours et gérer leur état (préparation en cours, service en cours, terminée)
- Gestion du menu : les employés pourront ajouter, modifier ou supprimer des boissons du menu
- Gestion des témoignages : les employés pourront modérer les témoignages des clients et les publier
- Statistiques de vente : les employés pourront consulter les statistiques de vente pour analyser les performances du Coffee Shop
- Gestion des contacts: les employés pourront consulter les entrées issues du formulaire de contact, gérer leur état et répondre aux questions depuis l'interface web.

Le site web sera conçu de manière responsive pour être accessible sur différents appareils, tels que les ordinateurs, les tablettes et les smartphones. Il utilisera une palette de couleurs chaleureuses et un design épuré pour refléter l'ambiance conviviale du Coffee Shop.
Les éléments suivants seront nécessaires pour sa mise en oeuvre:
- Docker (images déjà configurées)
- Laravel
- PHP 8.2 ou ultérieur
- MariaDB (déjà présente dans le Docker)
- un système d'authentification pour les salariés
- un gestionnaire de version (Git en local, je créerai un dépôt GitHub associé ultérieurement)
